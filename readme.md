# Biznesowa Kompania Serwerowa - Projekt Node.js

## Opis projektu

Projekt "Biznesowa Kompania Serwerowa" to aplikacja webowa stworzona w oparciu o Node.js, Express i MongoDB. Aplikacja umożliwia użytkownikom rejestrację, logowanie, oraz składanie zleceń w grze. Dzięki wykorzystaniu MongoDB dane przechowywane są w bazie danych, co umożliwia łatwe zarządzanie użytkownikami oraz zleceniami.

## Technologie

- Node.js
- Express
- MongoDB
- EJS (szablony)
- Bcrypt (do haszowania haseł)
- Docker (konteneryzacja)
